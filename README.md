# Windows Network Adapter Health Toolkit

A read-only PowerShell toolkit for network adapter configuration and health reporting.

## Features

- Adapter status and link speed
- IPv4, gateway, and DNS configuration
- Driver and interface context
- CSV, JSON, and HTML reports

## Run

```powershell
powershell.exe -ExecutionPolicy Bypass -File .\Windows_Network_Adapter_Health_Toolkit.ps1
```

## Safety

Read-only reporting only. No network settings are changed.
